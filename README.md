# 🎓 Student Management System (Python)

A simple **Student Management System** built using Python. This project allows users to manage student records through a command-line interface.

---

## 🚀 Features

* ➕ Add a new student
* 👀 View student details using roll number
* ✏️ Update existing student information
* ❌ Exit the program

---

## 🛠️ Tech Used

* Python
* Object-Oriented Programming (OOP)

---

## 📚 How It Works

* The system stores student data (name, marks, roll number) in a list.
* Users can interact with the system using a menu-driven interface.
* Each operation (Add, View, Update) is handled using class methods.

---

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/Shahid-fx-Trader/-Student-Management-System-Python-
```

2. Navigate to the project folder:

```bash
cd -Student-Management-System-Python-
```

3. Run the program:

```bash
python main.py
```
